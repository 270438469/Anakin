# ============================================================================
# 配置：监控地址和标签
# ============================================================================
# 如何新增地址和标签：
# 1. 在下面的 LABELS 字典中添加新条目，格式为：
#    "钱包地址": "标签"
# 2. 例如新增一个地址：
#    "0x1234567890abcdef1234567890abcdef12345678": "新地址标签"
# 3. 保存文件后，监视器会自动加载所有地址（无需修改其他代码）
# 
# 移除地址只需从字典中删除对应条目即可。
# ============================================================================
